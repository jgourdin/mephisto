# Installer Méphisto 😈 (guide pour les amis)

Extension pour le jeu WikiMasters. ~2 minutes à installer. Il faut **Google Chrome** (ou Arc, Brave, Edge — tous basés sur Chrome).

## Étapes

1. **Télécharge** le fichier `mephisto-extension.zip` que je t'ai envoyé.
2. **Décompresse-le** (double-clic sur Mac ; clic droit → « Extraire tout » sur Windows). Tu obtiens un dossier `mephisto`.
   - ⚠️ Garde ce dossier quelque part de stable (pas dans « Téléchargements » que tu vides) — l'extension pointe dessus en permanence.
3. Dans Chrome, va sur la page des extensions : tape **`chrome://extensions`** dans la barre d'adresse et Entrée.
4. En haut à droite, active **« Mode développeur »**.
5. Clique **« Charger l'extension non empaquetée »** (en haut à gauche).
6. Sélectionne le dossier **`mephisto`** décompressé → Ouvrir.
7. C'est bon ! L'icône du démon apparaît. Épingle-la (icône 🧩 puzzle en haut à droite → épingle Méphisto).

## Utiliser

- Va sur le jeu (`wiki-masters.com`). Un **bouton rond avec le démon** apparaît en bas à droite = le tableau de bord.
- Les réglages (activer l'auto-open, l'auto-bid…) sont dans ce panneau **ou** dans le popup (clic sur l'icône du démon dans la barre Chrome).
- **Tout est désactivé par défaut.** Coche « Automation active » pour démarrer, et laisse « Mode dry-run » coché tant que tu veux juste tester (ça simule sans agir).

## Bon à savoir

- Chrome peut afficher « Désactiver les extensions en mode développeur » à chaque démarrage — c'est normal pour ce type d'install, clique juste « Conserver ».
- Aucune donnée ne quitte ton ordi : l'extension lit seulement ta session du jeu, déjà ouverte dans le navigateur.
- Automatiser son compte de jeu se fait à ton propre risque (comme tout outil de ce genre).
